**uHomes** is a powerful but **simple-to-use** home management system with **GUI menu layouts**. Players can **set, teleport to, and manage several homes** with **visual confirmation dialogs and action bar notification**.


**Major Features:**


Nice GUI layout with clean home slots


5-second teleportation timer (cancels if moved by player)


Secure home deletion with confirmation dialogs


Permission-controlled home limits (1-7 homes)


Server restart persistent home storage


**Commands:**


**/homes** - Displays the homes management GUI


**/home** - Teleports to your default home (Home 1)


**Permissions:**


**uhomes.maxhomes.X** - Grants X amount of homes (replace X with 1-7)

Example: uhomes.maxhomes.5 grants 5 homes


Default home cap is configurable (default: 3 homes)


**Usage:**


Set a home by clicking a Lime Dye in the GUI


Teleport by clicking the corresponding Bed icon


Remove homes by clicking Red Dye and accepting


Higher ranks can be granted more homes via permissions


The plugin auto-saves all home locations and displays clear visual indication in the way of colored GUI elements and action bar messages. Perfect for survival servers who wish to have a tidy home system without flooding players with commands.
